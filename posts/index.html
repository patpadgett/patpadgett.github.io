Awesome Setup

I use these tools and related themes/plugins on my workstations.

  • i3 window manager
  • stormssh ssh config manager
  • tmux with tpm plugin manager, maglev theme, and tmuxinator for managing sessions
  • zsh with oh-my-zsh, bullet-train theme, and various plugins (git)
  • vim with vundle package manager, and various packages like vim-airline and powerline (see below)
  • powerline with powerline-patched Inconsolata font
